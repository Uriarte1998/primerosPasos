** git init inicializa el proyecto en git
Crea la carpeta .git -> administra los cambios del codigo
Es una carpeta oculta
** git status. Sirve para visualizar que archivos estamos trabajando

** git add <nombre> para agregar archivos al area de trabajo 
** git commit 
Se abre una ventana, se pone el mensaje, esc + ":wq" para salir 
** git log. Veo la informacion de un commit especifico
** git checkout -- <nombreArchivo> -> para volver a un estado anterior, si no se hizo un snapshot
** git diff <archivo> -> ve las diferencias
